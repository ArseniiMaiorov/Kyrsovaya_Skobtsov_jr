Шпаргалка: валидация данных
- `load_labeled_dataset(config)`: загрузка размеченного набора + проверка `Class` и значений меток.
- `load_unlabeled_dataset(config, expected_features)`: загрузка неразмеченного набора + проверка совпадения признаков.
- `load_datasets(config)`: согласованная загрузка обоих наборов.
- Дополнительно проверяются: число признаков (`49`), числовой тип признаков, правило `drop_all_nan_rows`.
- Допустимые метки на текущем этапе: `{0, 1, 2}`.
