# Спецификация предобработки (Этап 3)

## Разбиение данных
- Метод: `time_order_windows`.
- Параметры: `train_ratio=0.7`, `val_ratio=0.15`, `test_ratio=0.15`, `random_state=42`.
- Принцип:
  - сначала временной ряд делится на непрерывные непересекающиеся участки `train/val/test`;
  - затем окна строятся отдельно внутри каждого участка;
  - это исключает утечку из-за перекрывающихся окон между split.

## Версия RAW
- Приведение признаков к `float32`.
- Заполнение пропусков медианой по train.
- Масштабирование не применяется.

## Версия IMPROVED
- Заполнение пропусков медианой по train.
- Клиппинг по квантилям `0.01 / 0.99` (границы считаются на train).
- Масштабирование `RobustScaler`, `fit` только на train.
- Применение к `val/test/unlabeled` только через `transform`.

## Контроль утечек
- Строки исходного временного ряда не пересекаются между `train/val/test`.
- Параметры предобработки считаются только на train.
