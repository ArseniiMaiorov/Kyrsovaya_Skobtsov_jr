# Спецификация последовательностей

## Выбранный режим
- Режим: `A` (`time_windows`)

## Параметры окна
- `T = 128`
- `stride = 32`
- `overlap = 0.75`

## Правило формирования
- Скользящее окно по времени внутри одного непрерывного ряда или сеанса.
- Базовая форма окна: `X_window.shape = (samples, 128, n_features)`.

## Соглашение по входу модели
- `Conv1D` применяется по оси `steps = 128`.
- Каналы соответствуют `n_features`.

## Ограничения против утечки
- Разделение `train/val/test` выполняется так, чтобы окна не переносили информацию из будущего.
